TwoOkNotes

A WPF-based Desktop Application for Handwritten Notes

View Source Code

About The Project

TwoOkNotes is a desktop application built with C# and WPF that allows users to create, save, and manage digital handwritten notes. The project was developed to explore modern desktop development patterns and provide a simple, intuitive tool for note-taking.

Key Features

  • Dynamic Pen Tools: Customizable pen settings, including color and thickness, along with an eraser tool.
  • File Management: Integrated system to save notes as files, reopen them for editing, and organize them into notebooks.
  • Clean Architecture: Utilizes the MVVM (Model-View-ViewModel) design pattern to separate UI, logic, and data, enhancing maintainability and testability.
  • Undo/Redo Functionality: Implemented a command stack to allow users to easily undo and redo their actions.

Technologies Used
