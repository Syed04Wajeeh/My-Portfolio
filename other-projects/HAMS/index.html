About The Project

The Healthcare Appointment Management System (HAMS) is an Android application designed for a telehealth clinic, addressing the needs of three distinct user roles: Patients, Doctors, and Administrators. Developed in a team environment, this project demonstrates a full-cycle development process from UML design to final implementation and testing.

Key Features

  • Role-Based Access Control: Separate functionalities and interfaces for Patients, Doctors, and Administrators, ensuring users only access relevant information.
  • Administrator Dashboard: Administrators can review and approve/reject registration requests from new patients and doctors.
  • Doctor Schedule Management: Doctors can set their available work shifts, manage upcoming appointment requests, and view past appointments.
  • Patient Appointment Booking: Patients can browse doctors by specialty, book available time slots, cancel upcoming appointments, and rate doctors after a consultation.
